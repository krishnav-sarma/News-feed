# 📰 Console-Based News Feeder

This is a simple console-based application that shows the **top 12 news headlines** from your selected category using a public news API.

---

## 📚 Categories

When you run the app, you'll see this menu:

Choose a news category:
1.Technology
2.Sports
3.Business
4.World
5.Entertainment
6.Environment
7.Food
8.Health
9Politics
10.Science
11.Tourism
Enter choice (1-11):

Java 8 or above
Internet connection
